# XMRI No-Hair Fisher Matrix 代码逻辑梳理

本文档对仓库中的核心 notebook（`2026_XMRIs_No_Hair_Theory_FM-归一化-另一组参数.ipynb`）做结构化说明，帮助快速理解“从物理参数到 Fisher 约束结果”的完整流程。

## 1. 目标与总流程

该 notebook 的目标是：
- 构建 XMRI 信号模型（轨道 + 天琴 A/E/T 通道响应 + 噪声）。
- 通过数值差分构建 Fisher Information Matrix (FIM)。
- 反演得到协方差矩阵，评估参数不确定度（重点关注无毛定理相关参数 `Q`）。
- 对差分步长与矩阵条件数做稳定性诊断，避免“数值上可算、物理上不可信”的结果。

高层数据流如下：
1. **定义常数和源参数**：黑洞质量、自旋、偏心率、观测时长等。
2. **构建时域信号**：按谐波叠加得到响应波形。
3. **计算内积与 SNR**：基于噪声谱做加权。
4. **计算导数与 FIM**：对每个参数做有限差分并累积内积。
5. **反演与误差评估**：SVD/pinv 获得协方差，输出参数误差。
6. **稳定性检查**：扫描步长、查看条件数与对角占优等指标。

## 2. 代码结构（按功能分层）

> notebook 中存在“同名函数/类的重复定义”（例如 `run_stability_diagnostic`、`FIM_Diagnostic_Tool`），后定义会覆盖先定义。阅读时建议以**靠后的定义**为准。

### A. 基础工具层

- `sinc(parameter)`：基础 `sin(x)/x`。
- `fittingfactor(s1, s2, t)`：两条信号的归一化匹配因子（积分内积归一化）。

这一层主要用于快速校验和辅助分析。

### B. 物理参数与采样配置层

- 主要给出银河系中心黑洞、轨道几何、初相位、观测时长、采样间隔等。
- 构建统一时间轴 `t`，后续所有波形和积分都在该网格上完成。

### C. 相位与响应建模层

- `phase(...)`：构造每个谐波对应相位，包含频率漂移项和多普勒调制项。
- `AET_intrinsic_waveform(...)`：将极化/几何投影映射到天琴 A、E、T 通道的响应分量。
- 坐标与角动量相关函数（如 `calculate_vec_L`、黄道到笛卡尔坐标变换函数）：用于确定轨道平面与视线几何关系，支撑响应计算。

### D. 噪声模型与内积层

- `SnA(f, L_TQ)`、`SnT(f, L_TQ)`：A/T 通道噪声功率谱。
- `inner_product(dh1, dh2, t, N, f_orbit)`：在指定谐波频率下，按噪声权重计算导数波形内积。

这一步决定了 Fisher 元素的权重尺度，也是数值稳定性的关键来源之一。

### E. Fisher 构建与误差反演层

- 主流程会对参数集合（`lamda_s`, `beta_s`, `f_orbit`, `e_ecc`, `spin`, `Q`, `M_BH`, `m_BD`, `iota`, `alpha0`, `gmma0`, `Phi0`）逐一做有限差分。
- 将每对参数导数做噪声加权内积，填充 FIM。
- 使用 `np.linalg.svd` / `np.linalg.pinv` 的稳健逆获取协方差矩阵。
- 对角线开方得到 `sigma_i`，用于参数精度结论（如 `sigma_Q`）。

### F. 诊断与可视化层

- `check_fim_stability(...)`：输出条件数、参数步长等健康检查信息。
- `FIM_Diagnostic_Tool`：系统化提供 SNR 检查、步长扫描、FIM 质量诊断。
- `run_stability_diagnostic(...)`：扫描不同步长，观察导数/FIM 元素收敛区间。
- 若干绘图单元：
  - 展示不同实验对比的 `sigma_Q` 柱状图/带误差棒图。
  - 导出步长扫描图，辅助挑选“既稳定又不被舍入误差污染”的差分步长。

## 3. 关键“归一化参数化”思想

notebook 中一部分流程采用了“`params_norm` 接近全 1 的归一化参数空间”，再映射到 `physical_constants`：
- 好处：不同量纲参数（频率、质量、角度）能在更统一的数值尺度上做差分。
- 风险：若映射和步长策略不一致，容易出现“某些参数过小步长导致舍入噪声、某些参数过大步长导致截断误差”的混合问题。

因此该项目把“步长扫描 + FIM 条件数检查”放在较核心的位置。

## 4. 推荐阅读顺序（最快理解路径）

1. 先看“参数字典”单元：明确每个物理量的定义与默认值。
2. 看波形主函数（谐波叠加 + A/E/T 响应）输入输出。
3. 看 `inner_product` 与噪声谱函数，确认 Fisher 权重怎么定义。
4. 看 FIM 主循环（参数扰动、导数、矩阵填充）。
5. 最后看 `check_fim_stability` / `FIM_Diagnostic_Tool`，理解如何判断结果是否可信。

## 5. 当前 notebook 的工程性注意点

- 存在重复定义（后者覆盖前者），建议后续整理为单一实现。
- 参数、计算、画图交织在一个 notebook 内，复现实验时易受执行顺序影响。
- 推荐下一步重构：
  - `src/waveform.py`：波形与响应；
  - `src/noise.py`：噪声谱与内积；
  - `src/fisher.py`：导数与 FIM；
  - `src/diagnostics.py`：稳定性分析；
  - notebook 仅保留“实验编排 + 出图”。

这样可以让“可复现、可测试、可协作”显著提升。
